PennyCooking - 让烹饪更简单、更有条理

欢迎使用 PennyCooking！这是一款专注于「食谱管理、食材记录、用餐记录」的智慧厨房应用，帮助你把做菜这件事从“临时想起”变成“随时可执行”。

📱 应用简介

PennyCooking 帮你把厨房里的信息整理清楚：有哪些食谱、家里有哪些食材、最近吃了什么、哪些快过期、下一顿做什么。你也可以和家人共享一个“家庭厨房”，一起维护食谱与食材库存。

✨ 核心功能

  • 🍳 食谱管理：创建/编辑食谱，记录食材、步骤与图片，按难度与分类管理
  • 🧊 食材/冰箱管理：记录库存与分类，减少重复购买与浪费
  • 📝 用餐记录：记录每天吃了什么，方便回顾与统计
  • 👨‍👩‍👧‍👦 家庭共享：创建家庭厨房，与家人共享食谱与食材（邀请码加入）
  • 🔑 食谱分享码：生成分享码，把食谱分享给朋友；输入分享码即可导入
  • 🏆 成就系统：通过日常使用解锁称号/勋章，记录你的成长

📖 使用指南（快速开始）

1. 创建或导入食谱

  • 新建食谱：添加标题、食材与步骤，保存即可
  • 导入食谱：在“导入/添加”入口输入 分享码，一键导入

2. 记录食材库存

  • 在“食材/冰箱”页面添加食材
  • 建议按分类记录（蔬菜/肉类/海鲜/调料等），便于查找与整理

3. 记录用餐

  • 在“用餐记录”中记录你当天吃了什么
  • 用于回顾饮食习惯，也方便以后复刻好吃的那一餐

4. 家庭共享（家庭厨房）

  • 创建家庭厨房后，你的食谱与食材库可以邀请家人一起维护
  • 家人通过邀请码加入后，可共同编辑共享数据

❓ 常见问题（FAQ）

1) 分享码是什么？安全吗？

分享码用于跨用户分享与导入食谱。导入后会在你的账号/本地生成一份新的食谱数据，便于你后续自由修改。

2) 换手机后数据会丢吗？

如果你使用登录与同步功能，换设备后登录即可恢复数据。若仅离线使用，则数据保存在设备本地（建议定期备份）。

3) 家庭共享里，谁能看到我的数据？

家庭共享只针对你加入的“家庭厨房”数据。你的个人数据与用餐记录默认仍归你个人管理（以应用内实际功能为准）。

4) 会员购买后怎么恢复？

在 iOS 上通常可以通过 “恢复购买 / Restore Purchases”（如果应用内提供）或重新登录同一 Apple ID 来恢复权益。

5) 如何取消订阅？

在 iPhone/iPad：

  • 打开 设置 → 点击你的 Apple ID → 订阅 → 找到 PennyCooking → 取消订阅

取消后在当前计费周期结束前仍可使用权益，具体以 Apple 的订阅规则为准。

💳 订阅与付费说明

PennyCooking 可能提供不同的会员选项（如月度/年度/永久）。价格以 App Store 展示为准。

  • 订阅会在到期前自动续订（除非提前取消）
  • 购买与续订由 Apple 处理并在你的 Apple ID 中管理

🔒 隐私说明

我们重视你的隐私与数据安全：

  • 仅在提供核心功能所需范围内处理数据
  • 登录与同步功能需要网络连接
  • 若你不使用登录与同步功能，部分数据可能仅存储在本机

如需隐私相关咨询或数据处理请求，请通过下方邮箱联系。

🧾 账户与数据删除申请

如你希望删除账户或清除云端数据，请邮件联系并提供：

  • 账号（如注册邮箱/手机号，按实际情况）
  • 你的请求（删除账户/删除云端数据/导出数据等）
  • 设备与应用版本信息（可选）

我们将在合理时间内处理你的请求。
